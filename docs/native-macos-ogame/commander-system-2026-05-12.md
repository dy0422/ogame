# 指挥官系统说明

本模块把 OGame 式军官思路改造成单机快节奏局的指挥官玩法：不做付费货币，不做限时卡池，所有进度都保存在本地存档里。目标是让舰队出航、战斗和远征多一层长期成长选择。

## 核心循环

1. 通过招募令抽取指挥官。
2. 新指挥官进入名册，重复抽取转化为对应碎片。
3. 使用训练数据提升等级。
4. 使用碎片升星。
5. 在舰队派遣时选择一名空闲指挥官，让该舰队获得加成。
6. 战斗和远征会给派驻指挥官提供经验。

## 稀有度和保底

当前稀有度分为普通、精英、史诗、传奇。十连招募至少出现精英级指挥官；长期未出传奇时，40 抽内会触发传奇保底。重复指挥官不会生成第二个本体，而是转化为碎片：

- 普通：5 碎片。
- 精英：10 碎片。
- 史诗：25 碎片。
- 传奇：50 碎片。

## 成长

训练数据按 100 点经验提升 1 级。等级上限随稀有度提升：

- 普通：20 级。
- 精英：30 级。
- 史诗：40 级。
- 传奇：50 级。

升星消耗碎片，当前成本为 20、40、80、160、320。升星和等级都会放大舰队加成，但所有加成都保持在单机快节奏可控范围内。

## 职能

- 舰队：提高攻击和航速，适合突袭、追秒和主力进攻。
- 工程：提高护盾、结构并降低油耗，适合长航程、护航和稳健推进。
- 采掠：提高货舱和掠夺收益，适合清理低风险目标。
- 科技：提供火控和护盾校准，适合混合舰队。
- 远征：提高远征收益并降低高风险事件概率，适合非 PVP 成长路线。

## UI 入口

侧边栏新增“指挥官”。该页面显示招募令、训练数据、累计招募、传奇保底、名册、经验、碎片、升星状态和当前派驻状态。舰队页面的派遣面板新增“指挥官”选择器，只显示空闲指挥官。

## 平衡边界

指挥官只增强舰队，不替代建筑、科技和舰船克制。每支舰队最多派驻一名指挥官；已出航指挥官不能同时带第二支舰队。招募资源来自本地玩法，不包含真钱、暗物质商店或限时付费入口。
